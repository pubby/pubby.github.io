Grindless Pokemon FireRed

A rom hack for impatient gamers.

Two things I dislike about classic Pokemon:

  1. Grinding
  2. Beating the game using only one Pokemon

This hack attempts to fix these problems by taking /u/Think0028's Exp. All patch and applying experience point scaling so that the game's difficulty remains balanced. The result is (hopefully) a game you can play from start to finish without having to farm for experience in tall grass.

In addition, this hack also include the 151 Pokemon patch from /vp/, which enables version exclusive Pokemon from Leaf Green to be found in the wild.

Download patch for American (U) 1.0 version
